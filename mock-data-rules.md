# Mock Data Rules

1. Format: `mock_data.json` only — one unified object per company, post-ingestion format
2. 15 companies total — 3 invested, rest mixed statuses
3. 5 deep-coverage companies with 8–12 interactions each
4. 10 pipeline companies with 2–4 interactions each
5. 4 data sources: Granola, Affinity, Slack, Gmail
6. VC team members from both Yellow and Project A — random realistic names, mix of both firms
7. Every interaction has at least one named VC participant and one named external participant
8. Founders explicitly named per interaction
9. Rich realistic content — full narratives not placeholders
